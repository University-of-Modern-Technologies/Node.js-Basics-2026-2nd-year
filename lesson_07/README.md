# Тема 7. Робота з Express.js: статичні файли, шаблони, форми, завантаження

Навчальний демо-проєкт для лекції. ES Modules + Express 5. Приклади роботи зі статичними файлами, EJS шаблонами, обробкою форм та завантаженням файлів.

## Встановлення

```bash
npm install
```

Усі сервери слухають **порт 3000**.

---

## Кроки

### 01 — Статичні файли

```bash
npm run static
```

`express.static()` для роздачі статичних файлів (HTML, CSS, JS, зображення). `res.sendFile()` для віддачі конкретних файлів. Обробка POST форми з `express.urlencoded()`.

### 02 — Шаблони EJS

```bash
npm run templates
```

Налаштування EJS як шаблонізатора через `app.set('view engine', 'ejs')`. `express-ejs-layouts` для спільного layout. Передача даних у шаблони через `res.render()`. Динамічні сторінки з даними з "бази даних".

### 03 — Обробка форм

```bash
npm run forms
```

Робота з формами: `express.urlencoded()` для парсингу тіла форми. Отримання даних через `req.body`. Обробка POST запиту та відгук користувачу.

### 04 — Завантаження файлів

```bash
npm run upload
```

`multer` для завантаження файлів. Налаштування сховища (`diskStorage`). Обробка `multipart/form-data`. Валідація типів та розмірів файлів. Збереження завантажених файлів на сервері.

---

## Як запускати

З кореня `lesson_07`:

```bash
npm run static      # статичні файли
npm run templates   # EJS шаблони
npm run forms       # обробка форм
npm run upload      # завантаження файлів
```

Для кожного прикладу відкрийте `http://localhost:3000` у браузері.
