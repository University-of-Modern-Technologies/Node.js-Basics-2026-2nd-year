## Основи Node.js – код із занять

Це репозиторій з прикладами коду, які ми пишемо на заняттях у межах дисципліни **"Основи Node.js"**.  
Тут лежать демки з уроків + частини мініпроєкту та домашок, згідно з офіційним планом курсу.

### Структура репозиторію

- `lesson_01` – Вступ до екосистеми Node.js.
- `lesson_02` – Базові модулі. Робота з файлами та директоріями.
- `lesson_03` – Event Loop та асинхронне програмування.
- `lesson_04` – Вступ до Streams. Домашня робота №1: **File Organizer CLI**.
- `lesson_05` – Створення веб‑сервера з нуля.
- `lesson_06` – Знайомство з Express.js.
- `lesson_07` – Шаблонізатори, обробка форм та завантаження файлів в Express.
- `lesson_08` – Проміжне оцінювання (квіз, без коду).
- `lesson_09` – Робота з базами даних за допомогою **Prisma ORM**. Домашня робота №2: **Дошка оголошень**.
- `lesson_10` – Проєктування та створення RESTful API на Express. Домашня робота №3.
- `lesson_11` – Stateful аутентифікація, сесії та cookie.
- `lesson_12` – Stateless аутентифікація, JWT + refresh tokens. Домашня робота №4.
- `lesson_13` – Розширення функціоналу бекенду. Домашня робота №5.
- `lesson_14` – Практична робота: тестування Node.js застосунків.
- `lesson_15` – Практична робота: WebSockets / Socket.io, комунікація в реальному часі.
- `lesson_16` – Практична робота: TypeScript та публікація застосунку.
- Консультація (може не мати коду в цьому репозиторії).

Фактично присутні уроки дивись по реально існуючих папках `lesson_XX` у корені.

### Технології

- **Node.js** (LTS, наприклад `>=18.x`).
- **Express.js** – побудова HTTP‑сервера та REST API.
- **Prisma ORM + SQLite** для роботи з базою даних у мініпроєкті.
- **JWT**, cookies, сесії – для аутентифікації.
- **WebSockets / Socket.io** – для реального часу.
- **TypeScript** – для типізованої версії застосунку.

Конкретний стек на кожному уроці дивись за `package.json` і файлами в папці відповідного `lesson_XX`.

### Як запускати приклади

1. Перейти в потрібний урок, наприклад:

   ```bash
   cd lesson_01
   ```

2. Встановити залежності:

   ```bash
   npm install
   ```

3. Запустити скрипт з `package.json` для цього уроку. Приклад для `lesson_01`:

   ```bash
   npm run dev
   npm start
   ```

Для кожної папки `lesson_XX` використовуй тільки ті скрипти, які реально оголошені в її `package.json`.
